Features

Rubberduck takes inspiration from modern-day IDE features to bring the VBE into this century with game-changing tooling for all VBA programming skill levels. From helping Joe in Accounting with finding worksheets that have code behind them, to freelance veterans importing VBA code from source control repositories, or hobbyist programmers learning the ropes of OOP and unit testing in a language they're comfortable with - the VBE will never feel whole again without Rubberduck!
